# Splunk SOC Alert Triage Project

## Overview
This project demonstrates hands-on SOC analyst investigation and alert triage using Splunk SIEM. The project focused on reviewing suspicious authentication activity, web traffic, failed login attempts, and indicators of compromise through log analysis and dashboard creation.

## Skills Demonstrated
- Splunk SIEM Investigation
- Alert Triage
- Threat Hunting
- Log Analysis
- SSH Login Monitoring
- Web Request Analysis
- Dashboard Creation
- MITRE ATT&CK Mapping

## Tools Used
- Splunk
- Linux Logs
- SPL Queries
- MITRE ATT&CK Framework

## Project Structure
- 01_Report — Final project report
- 02_Screenshots — Investigation evidence and dashboard screenshots
- 03_Queries_Commands — SPL queries used during analysis
- 04_Dashboard — Splunk dashboard visuals
- 05_Readme — Project documentation
- 06_Resume_Bullets — Resume-ready project bullet points

## Key Findings
The investigation identified suspicious failed SSH login attempts, abnormal web requests, attacker IP activity, and potentially malicious behavior patterns through Splunk log analysis.

- T1078 — Valid Accounts
- T1059 — Command and Scripting Interpreter
