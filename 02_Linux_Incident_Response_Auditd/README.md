# Linux Incident Response and Auditd Analysis

## Overview
This project demonstrates hands-on Linux incident response, threat hunting, and audit log investigation techniques using native Linux logging utilities and Auditd analysis. The investigation focused on identifying suspicious processes, authentication activity, command execution history, and potential persistence mechanisms.
